# Triage Labels

This repo uses the default five-role label set for issue triage.

| Role | Label | Description |
|------|-------|-------------|
| Needs evaluation | `needs-triage` | Maintainer needs to evaluate the issue |
| Waiting on reporter | `needs-info` | More information is needed from the reporter |
| AFK-ready | `ready-for-agent` | Fully specified — an agent can pick this up with no human context |
| Needs human | `ready-for-human` | Needs human implementation or judgement |
| Won't fix | `wontfix` | Will not be actioned |

No custom labels are currently defined. If the project grows or a specific tool area needs its own label, add it here.
