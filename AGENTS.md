Single-context layout: `CONTEXT.md` at the repo root + `docs/adr/` for architectural decisions. See `docs/agents/domain.md`.

### Project overview

ToolsAulia is a client-side only (100% browser) tool collection built with Astro 5 + Tailwind CSS + TypeScript. All file processing happens locally — no data is ever sent to a server. PWA-enabled for offline use. Indonesian-first with English i18n support.
